# Upskill GCC 🚀

Upskill GCC is an AI-powered career acceleration platform designed to help expats and students become more employable in the Gulf Cooperation Council (GCC) job market.

Built to solve the challenges many professionals face when adapting their CVs, LinkedIn profiles, and skillsets for GCC recruiters and employers.

## 🌍 Live Demo
https://upskill-gcc.netlify.app

---

## ✨ Features

### AI Career Tools
- **GCC CV Optimizer** – Tailors CVs for GCC ATS/recruiter expectations  
- **LinkedIn Optimizer** – Enhances LinkedIn profiles for stronger visibility  
- **Skill Gap Analyzer** – Identifies missing skills for target roles  

### User Experience
- Authentication & protected dashboard  
- AI generation history tracking  
- Credits-based usage system  
- PDF export/download functionality  
- Profile management/editing  
- Contact & feedback system  
- Coming soon bootcamp request portal

## 🧠 Why I Built This

After moving to Doha, I personally experienced how difficult it was adapting to the GCC job market.  
Despite having skills, I struggled because my CV and positioning were not aligned with regional expectations.

Upskill GCC was created to solve that exact problem.
